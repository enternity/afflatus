# Long common sequence
> With string $a = (a_0,a_1, \ldots, a_x)$ and $b= (b_0,b_1, \ldots, b_x)$

$lcs(a_i, b_j) =  
  \begin{cases}
 n/2 & \quad  \text{if } n \text{ is even}\\
  -(n+1)/2 & \quad  \text{if } n \text{ is odd}
  \end{cases}$
